# Bài tập về nhà
Bài 1. Trang giới thiệu bản thân

        Trang HTML cần có:

        Họ tên, MSSV, ảnh chân dung.

        Một đoạn văn mô tả sở thích cá nhân.

        Danh sách các môn đã học trong năm 2024 (dùng thẻ <ul> hoặc <ol>).

        Liên kết tới một trang web yêu thích (dùng thẻ <a>).

Bài 2. Trang thông tin lịch học
        Trang HTML cần có:

        Họ tên, MSSV, ảnh chân dung.

        Bảng thời khóa biểu gồm: Tên môn học, Tên giảng viên, Ngày giờ học, Phòng học.

        Tiêu đề cho bảng (dùng thẻ <caption>).

        Đoạn mô tả ngắn ở cuối bảng, thể hiện cảm nhận về lịch học (dùng thẻ <p>).

        Yêu cầu chung
        Không sử dụng CSS hoặc JavaScript; chỉ dùng HTML.

        Phải sử dụng cấu trúc chuẩn HTML (semantic HTML).

        Mỗi file HTML có ít nhất 3 dòng comment (<!-- ... -->) để giải thích phần code quan trọng.

        Có header (ví dụ: tên trang) và footer (ví dụ: MSSV + ngày tạo).

        Sử dụng dữ liệu thật: họ tên thật, MSSV thật, ảnh thật, lịch học thật.

---Cách nộp bài
        Mỗi bài nộp trong một file .zip gồm:

        Mã nguồn HTML.

        Ảnh (chân dung và các ảnh dùng trong bài, nếu có).

        File .txt hoặc .docx mô tả cách tạo trang (5–7 câu).
